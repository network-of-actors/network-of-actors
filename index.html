Welcome

This webpage revolves around the network of actors and is made as part of the final project in the course 02805 Social graphs and interactions at the Technical University of Denmark (DTU) in the fall 2020. The web page contains an overview of the network of actors along with the results of different analyses performed on the network. For the technical details behind the analyses, we refer to the Explainer notebook. The data set and other files used to create this webpage can be found on this github.

Data

The data set is constructed specifically for this project and consists of a movies data file, and an actors data file. The data extraction is based on the list of the 1000 highest user rated movies on IMDB. Furthermore, a list of sentiments of words is downloaded and used to compute average sentiements of the movies. Based on the IMDB list we have created a data file of movies which after data cleaning consists of 862 rows and 15 attributes, which are specified below:
  • MovieID: Unique IMDB id of movie.
  • Title: Original title of movie.
  • ReleaseYear: Year movie was released.
  • Decade: Decade movie was released in.
  • Country: Main country movie was produced in.
  • Budget: Estimated budget of movie
  • Income: Cumulative worldwide gross of movie
  • ActorIDs: List of unique IMDB ids of actors that played in movie.
  • ActorNames: List of names of actors that played in movie.
  • Directors: Directors of movie.
  • Creators: Creators of movie.
  • Genres: List of genres the movie is categorized under.
  • PlotSynopsis: Plot synopsis of movie.
  • Keywords: List of keywords for movie.
  • Document: Tokens of cleaned plot synopsis and keywords.
From the total list of actors that have played in these movies a data file of actors is created, which consists of 28,862 rows and 37 attributes, which are specified below.
  • ActorID: Unique IMDB id of actor.
  • ActorName: Full name of actor.
  • Birthday: Birthday of actor.
  • BirthPlace: City, state and country, where actor was born.
  • BirthCountry: Country, where actor was born.
  • MovieIDs: List of unique IMDB ids of movies the actor has played in.
  • MovieIDs_xxx: Attribute for each decade xxx containing list of unique IMDB ids of movies the actor has played in in respective decade. Total of 10 attributes.
  • MovieIDs_yyy: Attribute for each genre yyy containing list of unique IMDB ids of movies the actor has played in in respective genre. Total of 21 attributes.
