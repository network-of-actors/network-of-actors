Welcome

This webpage is made as part of the final project in the course 02805 Social graphs and interactions at the Technical University of Denmark (DTU) in the fall 2020. The Python notebook, data and other files used to create this webpage can be found on this github.


Data

The data set is constructed specifically for this project and consists of a movies data file, and an actors data file. The data extraction is based on the list of the 1000 highest user rated movies on IMDB.
